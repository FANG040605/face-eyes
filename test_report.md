## 三、性能测试

### 3.1 测试方法

使用 `test_performance.py` 脚本进行自动化性能测试：

```bash
cd d:\ai-faces-practice\face_detection_app
d:\ai-faces-practice\venv\Scripts\python.exe test_performance.py
```

### 3.2 测试项 1：单次人脸识别响应时间

**测试内容**: 连续请求 5 次图片识别接口，测量响应时间

**测试结果**:

| 次数 | 响应时间 |
|------|----------|
| 第1次 | 2.97 秒 |
| 第2次 | 3.00 秒 |
| 第3次 | 2.92 秒 |
| 第4次 | 2.81 秒 |
| 第5次 | 2.82 秒 |

**统计结果**:

| 指标 | 数值 |
|------|------|
| 平均响应时间 | **2.90 秒** |
| 最快响应 | 2.81 秒 |
| 最慢响应 | 3.00 秒 |
| 是否达标 | ✅ 是（< 3 秒） |

### 3.3 测试项 2：并发人脸识别处理能力

**测试内容**: 同时发送多个请求，测试并发处理能力

**测试结果**:

| 并发数 | 总耗时 | 平均请求时间 | 成功率 |
|--------|--------|--------------|--------|
| 2 | 3.31 秒 | 2.99 秒 | 2/2 |
| 3 | 3.95 秒 | 3.29 秒 | 3/3 |
| 5 | 6.61 秒 | 4.80 秒 | 5/5 |

**测试结论**: 系统支持并发处理，成功率 100%

### 3.4 测试项 3：人脸注册响应时间

**测试内容**: 连续注册 3 次人脸，测量响应时间

**测试结果**:

**统计结果**:

| 指标 | 数值 |
|------|------|

| 是否达标 | ✅ 是（< 5 秒） |

### 3.5 测试项 4：人脸库查询响应时间

**测试内容**: 查询人脸库列表，测量响应时间

**测试结果**:

### 3.6 性能测试总结

| 测试项 | 平均响应时间 | 是否达标 |
|--------|--------------|----------|
| 单次人脸识别 | **2.90 秒** | ✅ |
| 并发2人识别 | 2.99 秒 | ✅ |

| 人脸库查询 | 2.06 秒 | ✅ |

### 3.7 性能指标对比（优化前后）

| 测试项 | 优化前 | 优化后 | 提升 |
|--------|--------|--------|------|
| 单次人脸识别 | 3.06 秒 | 2.90 秒 | +5.2% |
| 首次识别响应 | 4.59 秒 | 2.97 秒 | +35.3% |
| 人脸注册 | 2.96 秒 | 2.64 秒 | +10.8% |

**优化措施**: 将编码进程数从 `max(2, cpu_count)` 调整为 `max(4, cpu_count)`
